## 프로젝트 참여멤버 조회하기

| 메소드 | 경로                         | 짧은 설명                                 |
| ------ | ---------------------------- | ----------------------------------------- |
| GET    | /project/team/{프로젝트_idx} | 프로젝트에 현재 참여하고 있는 멤버들 조회 |

### 요청 헤더

```
authorization : token
```

#### 조회 성공

### 응답 바디

#### 세부사항 가져오기 성공

```json
{
    "message": "success",
    "data": [
        {
            "profile_url": "https://kb-today.s3.ap-northeast-2.amazonaws.com/1541789189792.png",
            "name": "트와이스",
            "id": "twice",
            "push_time": "13:00:00",
            "countFollower": 2,
            "countFollowing": 3
        }
    ]
}
```

#### 해당 사용자의 정보가 없을 때

```json
{
    "message": "Get data fail"
}
```

#### 유효하지 않은 token일 때

```json
{
    "message": "access denied"
}
```
